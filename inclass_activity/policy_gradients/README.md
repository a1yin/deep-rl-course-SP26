# Policy Gradients Implementation

Fill in the TODO sections in the `policy_gradients_assignment.ipynb` to implement each function.

## Requirements

- **PyTorch** (CPU version is okay)

If you don't have PyTorch on your laptop or local machine, you can use **Google Colab** instead.

## How to run on Colab

1. Upload `policy_gradients_assignment.ipynb` to Colab.
2. Upload `test_policy_gradients.py` via the **Files** tab on the left (folder icon). Click the upload icon; the file will appear in the default directory.

Then run the notebook cells and use the tests to check your implementation.
