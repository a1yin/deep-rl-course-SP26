# Bank Heist (text adventure)

In this game you break into an empty bank: pick up a sticky note in the lobby that reveals the vault PIN, enter it on the corridor keypad to open the vault door, grab the cash, and escape to the getaway alley to win. Any wrong codes trigger a silent alarm (game over), and going out the wrong exit in the lobby will lead you to be arrested. I chose a small heist scenario because it maps cleanly onto the assignment structure and I was inspired by a game called one-handed robber which I thought was funny.
